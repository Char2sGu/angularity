The `@angularity/core` package defines core concepts and offers utilities that are shared across all the other Angularity packages.

```sh
npm i @angularity/core
```

## Signature Symbols

Dependency Injection Utilities:

- `provide`
- `provideMulti`
- `injectRef`

Error Handling:

- `Exception`

Type Gymnastics:

- `$type`
- `TypeContainer`

RxJS Operators:

- `mapToVoid`
- `pickType`
